## 重大事件：Hadoop 3.5.0 正式发布

**HADOOP-19814** - Release Apache Hadoop 3.5.0 于2026年4月17日正式解决，标志着3.5.0版本发布。

### 3.5.0 核心特性

1. **Java 17运行时全面支持** - 包括JDK17 Profile、JPMS选项、单元测试修复等20+个子任务
2. **NameNode细粒度锁机制** - FS锁和BM锁分离，显著提升并发能力
3. **从Standby节点一致性读取** - 分担Active NameNode压力
4. **移除WritableRPCEngine** - 简化RPC引擎架构
5. **Jetty 12.x升级** - 从Jetty 9升级到12，解决多个安全漏洞
6. **javax到jakarta迁移** - 适配Jakarta EE规范
7. **移除sun.misc.Unsafe** - 提升安全性和Java版本兼容性

## 关键趋势分析

### 1. Java 17 全面落地

本次最大的主题是**Java 17运行时支持的全面完成**，涉及HADOOP-18887、HADOOP-17177及其20+子任务。包括：
- Maven JDK17 Profile
- JPMS模块选项
- 单元测试适配
- KMS/argLine/JDiff等组件适配
- Dockerfile升级到Ubuntu Noble(24.04)

### 2. HDFS 稳定性持续增强

- 修复多个**无限循环**和**NPE**问题（HDFS-17906、HDFS-17893、HDFS-17590）
- **EC相关**问题突出：DFSStripedInputStream异常、StripedBlockChecksumReconstructor连接泄漏
- RBF功能持续完善：多挂载点优化、trash路径删除修复
- 锁优化仍是重点：BM锁持有时间优化、ExcessRedundancyMap锁语义改善

### 3. YARN 走向现代化

- **cgroup v2支持**（YARN-11669）是重要演进，适配现代Linux容器运行时
- RM稳定性修复：ConcurrentModificationException导致RM崩溃
- 容器化增强：灵活目录权限、Docker清理改进

### 4. 云存储持续改进

- S3A：线程泄漏修复、KMS加密上下文支持、条件写入增强
- ABFS：FNS-Blob兼容性修复、迁移工具改进
- Wasb到ABFS迁移配置脚本支持

### 5. 构建与CI现代化

- 从Jenkins迁移到GitHub Actions
- OWASP依赖检查插件修复
- Docker构建平台升级

## 推荐行动

1. **升级到Hadoop 3.5.0**：获取Java 17支持、NameNode细粒度锁等核心特性
2. **关注cgroup v2进展**：如使用容器化部署，需关注YARN-11669的完成进度
3. **评估EC集群稳定性**：HDFS-15413、HDFS-17902等EC相关Bug需关注修复版本
4. **ABFS用户注意**：HDFS-17898（Blocker级别）影响FNS Blob迁移，需尽快升级
5. **S3A用户注意**：HADOOP-19862线程泄漏问题需关注修复
6. **监控DataNode无限循环**：HDFS-17906影响DataNode稳定性，需确认是否受影响
